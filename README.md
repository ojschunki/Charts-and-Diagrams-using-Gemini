# Charts & Diagrams using Gemini  
> As part of the ***CAP6640 Final Project***

Notebook-based project exploring how Gemini can generate charts and diagrams from natural language prompts. This project focuses on converting unstructured input into structured visual outputs within a lightweight, experimental environment.

---

## Project Objective

The goal of this project is to:
- Use Gemini to interpret natural language prompts
- Generate structured outputs for charts and diagrams
- Experiment with transforming text into visualization-ready formats
- Validate the feasibility of AI-driven visualization workflows

This project emphasizes **prompt engineering, structured output generation, and rapid prototyping**, rather than building a full application.

---

## Dataset

**Source:**  
[ChartQA via HuggingFace](https://huggingface.co/datasets/HuggingFaceM4/ChartQA)  

---

## How to Run

1. clone repo
2. create `.env` file
3. add this line (will possibly need your own API key)
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. can now run cells on the notebook
